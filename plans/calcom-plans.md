# Cal.com Plans

Cal.com offers four pricing tiers ranging from a free individual plan to enterprise with custom pricing. All plans include unlimited event types and calendar connections. API access is available on Teams and higher plans.

## Free

- 1 user
- Unlimited event types
- Unlimited calendar connections
- Cal Video (built-in video conferencing)
- Basic booking pages
- No API access

**Price:** $0/month

## Teams

- All Free features
- Round-robin scheduling
- Routing forms
- Booking analytics
- Remove Cal.com branding
- API access (120 requests/minute standard)
- Collective event types
- Fixed round-robin

**Price:** $12/user/month (billed annually) or ~$16/user/month (monthly billing)

## Organizations

- All Teams features
- Company subdomain (yourcompany.cal.com)
- SAML SSO + SCIM provisioning
- SOC 2 Type II compliance
- HIPAA compliance
- ISO 27001 compliance
- Organization-wide admin controls
- API access (up to ~200 requests/minute on request)

**Price:** $28/user/month (billed annually) or ~$37/user/month (monthly billing)

## Enterprise

- All Organizations features
- Custom pricing
- Priority support
- Dedicated database
- Higher SLAs
- EU data hosting option
- API rate limits up to 800 requests/minute
- Dedicated account manager

**Price:** Custom — contact Cal.com sales

## Self-Hosted (cal.diy)

Cal.com is fully open source. The `cal.diy` repository provides a version with no enterprise features locked behind a license — everything works out of the box without a Cal.com account. Self-hosters manage their own infrastructure and rate limits.

**Price:** Free (infrastructure costs apply)
